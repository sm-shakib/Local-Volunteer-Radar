# 🌐 Local Volunteer Radar
**A Community Volunteer Matching Web Platform**

Local Volunteer Radar connects skilled volunteers with nearby community events through skill-based matching and location aware filtering replacing slow, manual recruitment with a unified, structured workflow.

🔗 **Live:** [local-volunteer-radar-f.onrender.com](http://local-volunteer-radar-f.onrender.com)

## ✨ Features

### 👤 Volunteer
- User registration and secure authentication
- Skill-based volunteer profiles
- Discover and filter volunteer opportunities
- Interactive Google Maps integration
- One-click event registration
- Personal volunteering dashboard
- Real-time messaging with organizers
- Google Calendar synchronization
- QR badge generation for verification
- AI Copilot assistance

### 🗂️ Organizer
- Event creation and publishing
- Volunteer application review system
- Event announcements
- Volunteer participation tracking
- Real-time communication tools
- Export volunteer list
  
### 🛡️ Admin
- User account management
- Event moderation and approval
- Platform analytics dashboard
- Monitoring and system oversight

---

## 🛠️ Tech Stack

- **Frontend:** React + Vite
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **Real-Time:** Socket.IO
- **Auth:** JWT · Google OAuth 2.0
- **AI:** Groq API (LLaMA 3.1)
- **Integrations:** Google Maps API · Google Calendar API · Cloudinary · Nodemailer
